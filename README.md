
# Skin Disease Detection Using CNN

## Project Overview
This project focuses on detecting skin diseases using a Convolutional Neural Network (CNN).
The complete workflow includes data preprocessing, model training, and performance evaluation using Python and PyTorch.

## Tools & Technologies
- Python
- NumPy, Pandas, Matplotlib
- PyTorch
- Google Colab

## Dataset
Due to its large size, the dataset is not included in this repository.

## Methodology
- Data preprocessing and normalization  
- CNN-based model design and training  
- Model evaluation using accuracy and loss metrics  

## Results
The model was evaluated using accuracy and loss metrics.
Ongoing experimentation and hyperparameter tuning are being conducted to improve model accuracy.
